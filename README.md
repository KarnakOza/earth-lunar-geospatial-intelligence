# earth-lunar-geospatial-intelligence

**SAR + Hyperspectral Fusion for Earth & Lunar Terrain & Mineral Mapping**

# Overview
This project presents a multi-sensor geospatial intelligence pipeline that integrates:

  - SAR (Synthetic Aperture Radar) -> structural information
  - Hyperspectral data -> mineral composition
The system is designed to work across Earth and lunar environment, demonstrating cross-domain transfer learning (Earth -> Moon) and multi-sensor fusion for terrain and mineral mapping.
